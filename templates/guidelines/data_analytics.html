Analytics Fundamentals

Basic Skills (Click to expand)

  • Mathematics: Basic statistics, probability, and linear algebra.
  • Programming: Python or R (Python is preferred for its versatility).
  • Excel: Basic to advanced functions, pivot tables, and data visualization.

Soft Skills (Click to expand)

  • Critical thinking and problem-solving.
  • Communication skills for presenting insights.
  • Attention to detail.

Mathematics & Statistics

  • Descriptive Statistics: Mean, median, mode, variance, standard deviation
  • Probability: Basic probability concepts, distributions (normal, binomial)
  • Inferential Statistics: Confidence intervals, hypothesis testing
  • Linear Algebra: Vectors, matrices (basics for data manipulation)

Tool Fundamentals

  • Excel: Formulas, functions, pivot tables, charts
  • SQL: Basic queries, joins, aggregations
  • Python: Basics of programming for data analysis
  • R: Alternative language for statistical analysis

Business Understanding

  • Problem Framing: Converting business questions into data questions
  • Key Performance Indicators (KPIs): Defining and tracking metrics
  • Business Metrics: Revenue, conversion rates, customer lifetime value
Data Collection Topics

SQL & Databases

  • Advanced SQL: Subqueries, window functions, CTEs
  • Database Types: Relational (MySQL, PostgreSQL), NoSQL (MongoDB)
  • Database Design: Understanding data models and schema

Web Scraping

  • HTML Basics: Understanding webpage structure
  • Scraping Tools: BeautifulSoup, Scrapy, Selenium
  • Legal & Ethical Considerations: Robots.txt, rate limiting

APIs & Data Formats

  • REST APIs: Endpoints, authentication, requests
  • Data Formats: JSON, XML, CSV
  • API Libraries: Requests (Python), httr (R)
Data Processing Topics

Data Cleaning

  • Missing Data: Identifying and handling null values
  • Outlier Detection: Statistical methods, visualization
  • Data Quality: Consistency, accuracy, completeness

Data Transformation

  • Feature Engineering: Creating new variables
  • Normalization & Scaling: Min-max, z-score
  • Encoding: Handling categorical variables

Data Manipulation Libraries

  • Pandas (Python): DataFrames, groupby, merge
  • dplyr (R): filter, select, mutate, summarize
  • Data Pipelines: Creating reproducible workflows
Data Visualization Topics

Visualization Principles

  • Data-Ink Ratio: Maximizing information, minimizing clutter
  • Chart Selection: Choosing the right visualization for the data
  • Color Theory: Effective use of color in visualizations
  • Storytelling: Crafting a narrative with data

Visualization Tools

  • Tableau: Interactive dashboards and visualizations
  • Power BI: Microsoft's business intelligence tool
  • Python Libraries: Matplotlib, Seaborn, Plotly
  • R Libraries: ggplot2, Shiny

Dashboard Design

  • Layout & Composition: Organizing visualizations effectively
  • Interactivity: Filters, drill-downs, tooltips
  • User Experience: Designing for the intended audience
  • Performance: Optimizing for speed and responsiveness
Advanced Analytics Topics

Statistical Analysis

  • Hypothesis Testing: t-tests, chi-square, ANOVA
  • Correlation & Regression: Linear relationships, multivariate
  • Time Series Analysis: Trends, seasonality, forecasting

A/B Testing & Experimentation

  • Experiment Design: Control groups, randomization
  • Sample Size Determination: Statistical power, confidence
  • Analysis: Interpreting results, statistical significance

Predictive Analytics

  • Supervised Learning: Regression, classification
  • Model Evaluation: Metrics, cross-validation
  • Feature Importance: Understanding model drivers
  • Model Interpretability: Explaining predictions to stakeholders
